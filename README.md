# Resident-OnCall-Scheduler

**An Algorithm to assist Cheif Residents determine the On-Call Schedule each month. An automation approach to quickly build a schedule given complex constraints.**
-  We include variables such as time-off/exceptions, resident teams and call composition.
-  Currently our solution is limited to specific scenarios and workflows. We plan to move this forward and expand on the features allowing the user to customize the constraints and tailor a schedule that fits their workflow.
-  This is meant to be a baseline to fullfill all the requirements and constraints set by the user. Further analysis is required and changes may be needed to get the perfect schedule.
